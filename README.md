# Descripción del Proyecto
Este proyecto consiste en el desarrollo del clásico videojuego **Snake (La Serpiente)** utilizando el lenguaje de programación **C++**. El jugador controla una serpiente que debe desplazarse por el mapa para recoger comida, aumentando su tamaño y puntaje, evitando colisionar con los bordes o consigo misma.

El juego será desarrollado aplicando conceptos fundamentales de programación aprendidos en la materia de **Programación 1**, incluyendo estructuras de control, funciones, arreglos, lógica algorítmica y manejo de entradas del usuario.

# Propósito del Proyecto
El propósito de este proyecto es aplicar de manera práctica los conocimientos adquiridos en la asignatura de **Programación 1**, mediante el desarrollo de un videojuego interactivo que permita fortalecer habilidades de lógica de programación, resolución de problemas y trabajo en equipo.

Además, el proyecto busca comprender la estructura de un programa completo en C++, implementando mecánicas de movimiento, puntuación y control de errores dentro de un entorno dinámico.

# Objetivos

# Objetivo General
Desarrollar un juego funcional de **Snake en C++**, aplicando conceptos básicos e intermedios de programación.

# Objetivos Específicos
- Implementar el movimiento de la serpiente.
- Diseñar un sistema de puntuación.
- Agregar comida aleatoria para el crecimiento de la serpiente.
- Detectar colisiones con bordes y consigo misma.
- Mejorar habilidades de programación en equipo.

# Funcionalidades del Juego
- Movimiento de la serpiente.
- Sistema de puntaje.
- Crecimiento progresivo de la serpiente.
- Generación aleatoria de comida.
- Finalización del juego por colisión.
